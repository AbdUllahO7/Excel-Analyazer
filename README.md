Excel Data Analyzer
A powerful web-based application for analyzing, visualizing, and transforming Excel data without leaving your browser.

Features
Data Management

Upload & Import: Support for Excel (.xlsx, .xls) and CSV files
API Connector: Connect to external APIs to import data
Multi-File Processing: Combine data from multiple files with advanced join operations
Recent Files: Quickly access recently uploaded files
Sample Datasets: Explore functionality with built-in sample data

Data Exploration

Interactive Data Table: Sort, filter, and search through your data
Data Cleaning: Handle missing values, remove duplicates, fix outliers
Data Transformation: Apply normalization, binning, and custom formulas
Column Management: Add, remove, rename, and modify columns

Data Visualization

Multiple Chart Types: Bar charts, line charts, pie charts, and scatter plots
Annotation Tools: Add notes and highlights to your visualizations
Comparative Analysis: Compare data across different dimensions with side-by-side charts
Export Options: Download charts as SVG or PNG

Advanced Analytics

Statistical Analysis: Get comprehensive statistics for your data
Intelligent Analysis: Automatically detect trends, correlations, anomalies, and predictions
Server-Side Processing: Handle large datasets with server processing options
Customizable Templates: Apply pre-defined analysis templates for common scenarios

Collaboration

Commenting System: Add comments and annotations to your analysis
Session Management: Save and restore analysis sessions
Collaboration Indicators: See who's working on the same data
Sharing Options: Share your analysis with others via link or email
Export Reports: Generate PDF or Word reports of your analysis

Getting Started
Prerequisites

Node.js 14.x or higher
npm or yarn

Installation

Clone the repository

bashCopygit clone https://github.com/yourusername/excel-data-analyzer.git
cd excel-data-analyzer

Install dependencies

bashCopynpm install
# or
yarn install

Start the development server

bashCopynpm run dev
# or
yarn dev

Open http://localhost:3000 in your browser

Technology Stack

Frontend: React, Next.js, Tailwind CSS, shadcn/ui
Data Processing: XLSX.js, Papa Parse
Visualization: Recharts
State Management: React Hooks
